Table 5.8 Membership operators in Python
Operator Description Example (Try in Lab)
in Returns True if the variable/value is found in the specified sequence and False otherwise
>>> a = [1,2,3]
>>> 2 in a
>>> '1' in a not in Returns True if the variable/value is not found in the specified sequence and False otherwise
