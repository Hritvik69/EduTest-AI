9.8.1 Single Row Functions
These are also known as Scalar functions. Single row functions are applied on a single value and return a single value. Figure 9.3 lists different single row functions under three categories — Numeric (Math),
String, Date and Time.
Math Functions accept numeric value as input and return a numeric value as a result. String
Functions accept character value as input and return either character or numeric values as output. Date and
Time functions accept date and time value as input and return numeric or string or Date and Time as output.
